Blocs Developer allows you to connect Wordpress site content with the Blocs desktop Mac application.

== Description ==

This plugin is intended for use with the website and Wordpress theme builder [Blocs For Mac](https://blocsapp.com/). It expands the Wordpress json API and adds additional endpoints for greater integration with the Blocs design environment. This plugin is only required during the development phase and is not required once a theme is exported from Blocs and installed into Wordpress. 

== Installation ==

It's easy to get started

1. Upload `blocs-developer` unzipped folder to the `/wp-content/plugins/` directory or goto Plugins > Add New and upload the blocs-developer zipped file.
2. Activate the plugin through the 'Plugins' menu in WordPress.
